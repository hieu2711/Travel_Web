# Travel_Web
Application for search and travel services
## Settings:
1. **Clone repository**:
   
      ```bash
   git clone https://github.com/hieu2711/Travel_Web
2. **Set up**:
   - Client:
     ```sh
     cd travelweb
     npm install
      ```
   - Server:
     ```sh
     Install Apache Netbeans
     Install Tom Cat
     run server
      ```
